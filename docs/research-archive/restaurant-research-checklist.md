# Restaurant-Recherche-Checkliste

## Zu recherchierende Daten pro Restaurant

### Basis-Informationen (existieren teilweise)
- [ ] **Website** - Offizielle URL
- [ ] **Adresse** - Vollständige Adresse (Straße, PLZ)
- [ ] **Telefon** - Vorwahl + Nummer
- [ ] **Öffnungszeiten** - Wochentage + Uhrzeiten
- [ ] **Zahlungsarten** - Bar, Karte, Apple Pay, etc.

### Menü & Spezialitäten
- [ ] **Food Menu** - Signature dishes, Portionsgrößen, Preise
- [ ] **Vegetarische Optionen** - Ja/Nein + Beispiele
- [ ] **Vegane Optionen** - Ja/Nein + Beispiele
- [ ] **Allergene** - Glutenfrei, Laktosefrei, etc.
- [ ] **Haus-Specials** - Gerichte, die man probieren muss

### Atmosphäre & Ambiente
- [ ] **Musik-Typ** - Stil, Lautstärke (leise/laut)
- [ ] **Crowd-Typ** - Zielgruppe (Lokale, Touristen, Business, Families)
- [ ] **Seating-Optionen** - Indoor, Outdoor/Terrasse, Bar
- [ ] **Raucher** - Raucherbereich vorhanden?
- [ ] **Reservierung** - Erforderlich? Empfohlen?

### Spezielle Infos
- [ ] **Pro Tips** - Insider-Tipps (Best time, What to order, etc.)
- [ ] **Fun Facts** - Interessante Hintergründe, Geschichte
- [ ] **Sprache** - Spricht das Personal Englisch?
- [ ] **WiFi** - Verfügbar? Kostenlos?
